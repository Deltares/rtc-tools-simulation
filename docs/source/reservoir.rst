Single Reservoir
================

This section documents the :py:class:`ReservoirModel` class,
which can be used to model a single reservoir.
An overview of the methods of :py:class:`ReservoirModel`,
including all the schemes to control the reservoir flow is given in :ref:`reservoir-api`.
Details of the Modelica file used by :py:class:`ReservoirModel`
can be found in :ref:`single-reservoir-model`.
We encourage users to first look at :ref:`examples-single-reservoir`
before looking at the details of :py:class:`ReservoirModel` in this section.
